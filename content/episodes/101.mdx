This episode covers how NoRedInk implements text editing for their application using Quill, a JavaScript rich text editing library, and a custom element. Brian Hicks show us an interesting look into how Elm only is not always they best option in a business context, but covers how the guarantees Elm gives us can be leveraged to still create a wonderful solution.
